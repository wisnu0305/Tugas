Daftar Profesi Cita-Cita sebagai web depeloper
Profesi Keterangan
Front-End Depeloper Fronte-end developer biasanya akan mengubah desain yang dibuat oleh perancang web menjadi antarmuka pengguna yang fungsional dan estetis menggunakan HTML, CSS dan JavaScript dalam proses mendesain halaman web
Back-End Developer Back-end developer membangun dan memelihara teknologi yang menggerakan komponen-komponen yang bersama-sama, memungkinkan sisi situs web yang muncul ditampilan pengguna dapat diakses
UI Depeloper Tugas UI developer adalah merancang dan membangun antarmuka perangkat kunak dengan memperioritaskan pengguna dan persyaratan bisnis di atas batasan dan kendala teknologi
UX Designer Desain UX berfokus pada interaksi antara pengguna manusia nyata dan produk layanan sehari-hari, seperti situs web, aplikasi, dan bahkan mesin kopi; menjadikan bidang UX Design sebagai disiplin yang sangat bervariasi, menggabungkan aspek psikologi, bisnis, riset pasar, desain, dan teknologi
